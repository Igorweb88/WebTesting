# Веб-тестирование

Проект **Веб-тестирование** предназначен для организации и проведения тестирования веб-приложений с упором на функциональное тестирование сервиса Gmail (отправка сообщений).

## Описание проекта

Проект включает:

- Подробные **тест-кейсы** для проверки функциональности веб-приложений.  
  [Открыть тест-кейсы](https://docs.google.com/spreadsheets/d/1jxIHkW8qRBmJPgPvn8PcvRz7PL185554kbUsRTMiimE/edit?usp=drive_link)

- **Чек-листы** для быстрого контроля ключевых функций.

- **Набор тестов Gmail**, с фокусом на проверке отправки сообщений.
