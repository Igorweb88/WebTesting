# Web Testing

The **Web Testing** project is designed for organizing and performing web application testing, with a focus on functional testing of the Gmail service (sending messages).

## Project Description

The project includes:

- Detailed **test cases** for checking the functionality of web applications.  
  [Open test cases](https://docs.google.com/spreadsheets/d/1jxIHkW8qRBmJPgPvn8PcvRz7PL185554kbUsRTMiimE/edit?usp=drive_link)

- **Checklists** for quickly verifying key features.

- **Gmail test suite**, focusing on testing message sending.
